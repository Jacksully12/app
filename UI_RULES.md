# UI Rules

## Results cards
Use this format:

```text
MODEL
0.5 HP • 0.37 kW • Phase S
Borewell Submersible
At 40.0 m: 1,257 LPH • +57 LPH
Page 10 • Size 1½” • TEXMO
```

## Details screen
Show:
- model header
- performance curve
- curve points used
- quick specs
- catalogue section
- Share WhatsApp / Share Image buttons
- Copy model number / Back buttons

## Share image
Customer-facing image only:
- Granpa branding
- recommended model
- HP / kW / phase / type
- selected head
- estimated output
- clean chart
- generated by Granpa Pump Selector
- no page number
