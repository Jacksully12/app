APP UI + FUNCTIONAL QA REPORT v5.1.0
===============================================

UI updates:
- Preserved Texmo main and detailed category dropdown.
- Brand-specific detailed categories remain available for Lubi and KSB.
- Reduced input, spinner and button heights.
- Added Change brand action.
- Improved minimum/maximum flow labels in range mode.

Functional controls:

- Collapsible result groups.

- Range results capped to 25 best low-power matches.
