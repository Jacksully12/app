Granpa Pump Selector Deep Logic QA - v3.6.0
========================================================

Issues found and fixed
- All pump types was previously mixed into one common result list; low-HP borewell items could hide openwell/monoblock/dewatering groups.
- Fixed: All pump types now shows results grouped by main category.
- Previous fallback sorting could allow a very wide ±50% low-HP option to hide a better ±10%/±20% match.
- Fixed: tolerance band priority is checked before HP; inside the same band, lower HP/kW is preferred.

Final fixed-flow result logic
- Specific pump type: show max 2 above/equal target flow and max 2 below target flow.
- All pump types: repeat the same max 2 above + 2 below logic separately for each main category.
- Bands: Best ±10%, Extended ±20%, Wide ±30%, Last option ±50%.
- Sort priority: band quality first, then lower HP, lower kW, closest flow, model name.
