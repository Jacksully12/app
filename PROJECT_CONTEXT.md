# Granpa Pump Selector — Project Context

Granpa Pump Selector is a native Android app that helps users find suitable pump models from a catalogue dataset.

## Current requirements
- App name: Granpa.
- Native Android Java app.
- Main search model field must stay at the top.
- Brand filter is removed from the main search screen.
- Results cards keep full technical details including page number.
- Model details keep page number and catalogue verification details.
- Share/download image is clean and customer-facing with no page number.
- Model details has two main actions only:
  - Share WhatsApp
  - Download Image

## Catalogue section
The catalogue section should show compact text from the catalogue page header:
`TEXMO | 4” Stainless steel Borewell Radial Flow Submersible Pumpsets / Single Phase 220 Volts, 3 Phase 380 Volts, 50 Hz`
